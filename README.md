# Self Growth 自我生成手机应用使用情况统计Android客户端
***

**自我生长：成为更好的自己**

## 简介
用户获取手机当前的正在使用的应用，将其上报到服务端

## 功能简介
客户端主要功能定位是：

- [x] 收集手机当前应用使用情况
- [ ] 登录集成
    - [ ] 手机号码一键登录
    - [ ] 网页端扫描登录
- [ ] 今日当前截止手机应用使用情况
- [x] 任务列表

核心功能主要是能收集当前手机正在使用的APP情况，上报到服务端

更多的管理和展示，目前规划到Web端
